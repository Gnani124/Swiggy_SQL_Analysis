# 📊 Swiggy Business Analysis using SQL

This project focuses on performing **business analysis on Swiggy order data** using SQL. The goal is to extract meaningful insights from raw data to support business decisions, improve customer experience, and optimize operations.

---

## 🚀 Project Overview

* Analyzed food delivery data using **SQL queries**
* Built insights on **orders, revenue, customer behavior, and trends**
* Used **data aggregation, joins, filtering, and date functions**
* Simulated real-world **business analyst scenarios**

---

## 🛠️ Tools & Technologies

* SQL (MySQL / PostgreSQL)
* Data Analysis
* Query Optimization
* Relational Databases

---

## 📂 Key Analysis Performed

* 📅 Monthly & yearly order trends
* 💰 Revenue analysis
* 🍽️ Most ordered items
* 📍 City-wise performance
* 👥 Customer behavior insights
* ⏱️ Peak order times

## 📈 Sample Insights

* Identified high-demand periods and peak hours
* Found top-performing restaurants and items
* Analyzed customer ordering patterns
* Evaluated revenue growth trends
